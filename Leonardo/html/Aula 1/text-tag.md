# Tag <p></p>

A tag '<p>' é usada para definir um parágrado em HTML. Ela é uma das tags mais comuns e essenciais para estruturar o conteúdo textual em uma página web

#### Sintaxe

''''html
<p> Este é um parágrafo de exemplo </p>

# Tag de Cabeçalho '<h1> a <h6>'

As tags de cabeçalho '<h1>' a '<h6>' são usadas para definir títulos e subtítulos em uma página HTML. A tag '<h1>' representa o título mais importante, enquanto '<h6>' representa o título menos importante.

##### Sintaxe 

'''html
<h1>Título Principal</h1>
<h2>Subtitulo</h2>
<h3>Sub-subtitulo</h3>
<h4>Título de Nível 4</h4>
<h5>Título de Nível 5</h5>
<h6>Título de Nível 6</h6>

## Tag '<strong></strong>'

A tag '<strong>' é usada para definir texto com ênfase forte geralmente exibido em negrito pelos navegadores

###### Sintaxe

''''html
<strong>Este texto está em negrito.</strong>
''''

## Tag '<small> </small>'

A tag '<small>' é usada para definir texto que deve ser exibido em um tamanho menor, geralmente usado para notas de rodapé ou informações adicionais

###### Sintaxe

'''html
<small>Este é um texto em tamanho menor</small>
'''
## Tag '</br>'

A tag '<br>' é usada para inserir uma quebra de linha no texto. Ela não possui uma tag de fechamento e é uma tag auto-fechável.

##### Sintaxe

'''html
Este é o primeiro parágrado .<br>
Este é o segundo parágrado após a quebra de linha
'''

## Tag '<em></em>'

A tag '<em>' é usada para definir texto com ênfase, geralmente exibido em itálico pelos navegadores

##### Sintaxe

'''html
<em> Este texto está em itálico</em>
'''

## Tag '<i></i>'

A tag '<i>' é usada para definir texto em itálico, geralmente usado para termos técnicos, nomes de obras ou palavras estrangeiras

## Sintaxe

'''html
<i>Este texto está em itálico</i>
'''
