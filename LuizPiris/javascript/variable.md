# JavaScript Variables

## Examples

1. **String Variable**
```javascript
Let name = "Luiz"; // A variável já foi definida

name = "João";     // Não precisa definir novamente a variável para mudar o valor, basta atribuir o novo valor
```

2. **Number Variable**
```javascript
Let age = 25;
```

3. **Boolean Variable**
```javascript
Let isActive = true;
```

4. **Array Variable**
```javascript
Let fruits = ["apple", "banana", "orange"];
```

6. **Const Variable**
```javascript

```
