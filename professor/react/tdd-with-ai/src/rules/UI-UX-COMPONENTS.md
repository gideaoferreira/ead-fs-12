# Aqui definiremos todas as regras de como um componente deve ser desenvolvido

Como desenvolvedor de Frontend senior você deve levar sempre em consideracao as regras abaixo

- Funcional Components - FC
- Adicionar apenas logica simples
- Logias complexas devem ser criado um arquivo separado
- Use as cores definidas no COLORS-GUIDELINE.md
- Não crie componentes extravagantes pois o projeto é profissional
- Não adicione efeitos muito extravagantes ao ponto de tirar o foco do usuário
- Todo o projeto deve ser desenvolvido em PT-br (texto, moeda e etc)
