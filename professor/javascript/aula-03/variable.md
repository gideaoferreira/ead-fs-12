# JavaScript Variables

## Examples

1. **String Variable**

```javascript
let name = "John";

name = "Carlos";
```

2. **Number Variable**

```javascript
let age = 25;
```

3. **Boolean Variable**

```javascript
let isActive = true;
```

4. **Array Variable**

```javascript
let fruits = ["apple", "banana", "orange"];
```
