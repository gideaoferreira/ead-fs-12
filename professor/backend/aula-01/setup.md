# Configuração do ambiente

## Dependencies

- Instalar o express: `npm install express`
- Instalar o cors: `npm install cors`

### Dependências para o banco de dados

- Instalar o mysql2: `npm install mysql2`
- Instalar o sequelize: `npm install sequelize`

## DevDependencies

- Instalar o nodemon: `npm install -D nodemon`

### Dependências de desenvolvimento para o banco de dados

- Instalar o sequelize-cli: `npm install -D sequelize-cli`
