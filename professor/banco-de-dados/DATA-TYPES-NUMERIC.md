# Numeric Data Types

[https://www.w3schools.com/mysql/mysql_datatypes.asp]

#### INTEGER(size)

- Armazena um número inteiro, com um tamanho máximo definido pelo usuário.

#### BOOLEAN

- Armazena um valor booleano, que pode ser verdadeiro (1) ou falso (0).

#### BIGINT

- Armazena um número inteiro grande, com um tamanho máximo de 9.223.372.036.854.775.807.

#### FLOAT(size, d)

- Armazena um número de ponto flutuante, com um tamanho total definido pelo usuário e um número de casas decimais definido pelo usuário.

#### DOUBLE(size, d)

- Armazena um número de ponto flutuante de precisão dupla, com um tamanho total definido pelo usuário e um número de casas decimais definido pelo usuário.
