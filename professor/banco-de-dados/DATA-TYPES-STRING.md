# String Data Types

[https://www.w3schools.com/mysql/mysql_datatypes.asp]

#### VARCHAR(size)

- Armazena uma sequência de caracteres de comprimento variável,
- com um tamanho máximo definido pelo usuário.
- O tamanho máximo pode ser de até 255 caracteres.

#### TINYTEXT

- Armazena uma sequência de caracteres de comprimento variável,
- com um tamanho máximo de 255 caracteres.
- Ele é usado para armazenar pequenas quantidades de texto,
- como descrições curtas, comentários, etc.

#### MEDIUMTEXT

- Armazena uma sequência de caracteres de comprimento variável,
- com um tamanho máximo de 16.777.215 caracteres.
- Ele é usado para armazenar grandes quantidades de texto,
- como artigos, postagens de blog, etc.

#### LONGTEXT

- Armazena uma sequência de caracteres de comprimento variável,
- com um tamanho máximo de 4.294.967.295 caracteres.
- Ele é usado para armazenar quantidades extremamente grandes de texto,
- como livros, documentos, etc.
