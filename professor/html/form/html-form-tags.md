# Tags html de formulários

Os formulários HTML são criados usando a tag `<form>`,
que pode conter vários elementos de entrada, como campos de texto,
botões, caixas de seleção, entre outros.

Abaixo estão alguns dos principais tags HTML usados em formulários:

- `<form></form>`: Define o início e o fim de um formulário HTML.
- `<input>`: Define um campo de entrada onde o usuário pode inserir dados.

  - Atributos comuns do `<input>`:
    - `type`: Define o tipo de entrada (ex: text, password, email, checkbox, radio, file, data, time, etc.).
    - `name`: Define o nome do campo de entrada.
    - `value`: Define o valor padrão do campo de entrada.

- `<label>`: Define um rótulo para um elemento de entrada.
- `<textarea>`: Define uma área de texto multilinha para entrada de dados.
- `<select>`: Define uma lista suspensa (dropdown) para seleção de opções.
  - `<option>`: Define uma opção dentro de uma lista suspensa.
