# Tags de interação HTML

As tags de interação em HTML são elementos que permitem aos usuários interagir com o conteúdo de uma página web. Essas tags são essenciais para criar formulários, botões, links e outros elementos interativos que melhoram a experiência do usuário.

## Principais Tags de Interação

### `<a>` - Link

A tag `<a>` é usada para criar links para outras páginas ou recursos. Ela pode ser configurada para abrir links em novas abas, baixar arquivos, entre outras funcionalidades.

```html
<a href="https://www.exemplo.com">Link abrindo na mesma página</a>
<a href="https://www.exemplo.com" target="_blank"
  >Link abrindo em nova página/aba</a
>
```

### `<button>` - Botão

A tag `<button>` é usada para criar botões clicáveis. Eles podem ser usados para
submeter formulários, executar scripts ou qualquer outra ação interativa.

```html
<button type="button" onclick="alert('Botão clicado!')">Clique Aqui</button>
```
