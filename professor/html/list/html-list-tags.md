# Lista html

Lista HTML é usada para agrupar itens relacionados em um formato ordenado ou não ordenado.
Existem dois tipos principais de listas em HTML: listas ordenadas e listas não ordenadas.

## Lista não ordenada

Uma lista não ordenada é usada quando a ordem dos itens não é importante.
Ela é criada usando a tag `<ul>` (unordered list) e cada item da lista é definido com a tag `<li>` (list item).

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

## Lista ordenada

Uma lista ordenada é usada quando a ordem dos itens é importante.
Ela é criada usando a tag `<ol>` (ordered list) e cada item da lista também
é definido com a tag `<li>`.

```html
<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ol>
```

## Lista de definição

Uma lista de definição é usada para definir termos e suas descrições.
Ela é criada usando a tag `<dl>` (definition list), onde cada termo é definido com
a tag `<dt>` (definition term) e cada descrição é definida com a tag `<dd>` (definition description).

```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
```

Essas são as principais tags usadas para criar listas em HTML. Elas ajudam a organizar o conteúdo de forma clara e estruturada.
